Latest Research Articals

FInC Flow: Fast and Invertible k × k Convolutions for Normalizing Flows (accepted: VISAPP'23)

We propose a k × k convolutional layer and Deep Normalizing Flow architecture which i.) has a fast parallel inversion algorithm with running time O(nk**2) (n is height and width of the input image and k is kernel size), ii.) masks the minimal amount of learnable parameters in a layer. iii.) gives better forward pass and sampling times comparable to other k × k convolution-based models on real-world benchmarks.

New

As With a parallel inversion approach, we present a k × k invertible convolution for Normalizing flow models. We utilize it to develop a model with highly efficient sampling pass, normalizing flow architecture.

paper and code

Remote sensing data clustering using autoencoder neural networks for lithological mapping (pre-print)

Remote sensing data clustering using autoencoder networks for lithological mapping.

New

As Auto-Encoder are able to model the complex high-dimensional distributions of real-world data, they offer a promising approach to address this challenge. In this work, we propose an anomaly detection method using AEs and clustering algorithms. We also compare our results with the Principal component analysis (PCA), Independent Component Analysis (ICA), and Minimum Noise Fraction (MNF) dimensionality reduction techniques known as multivariate statistical methods that convert a set of observed and correlated input variables into uncorrelated or independent components.

paper and code

R2I-rPPG: Robust Region of Interest selection for remote PPG (submitted: Illinois College of Emergency Physicians (ICEP Conference-22))

remote photoplethysmography (rPPG)

New

The rapid expansion of telemedicine during the COVID-19 pandemic has highlighted some important limitations of virtual care, such as the lack of access to patient vital signs during the visit. Estimation of specific vital signs through remote video-based photoplethysmography (rPPG) has been demonstrated. We are investigating how to estimate heart rate remotely.

CInC Flow: Characterizable Invertible 3×3 Convolution (accepted: 4th Workshop on Tractable Probabilistic Modeling, UAI 2021)

Invertible 3x3 Convolution

New

We study conditions such that 3x3 CNNs are invertible, allowing them to construct expressive normalizing flows. We derive necessary and sufficient conditions on a padded-CNN for it to be invertible.

Automated Seed Quality Testing System using GAN & Active Learning (accepted: PReMI'21)

solving the problem of labelling the images and dataset imbalanced

New

We build a novel seeds image acquisition setup, which captures both the top and bottom views.Dataset collection for this problem has challenges of data annotation costs/timeand class imbalance. We address these challenges by i.) using a Conditional Generative Adversarial Network (CGAN) to generate real-looking images for the classes with lesser images and ii.) annotate a large dataset with minimal expert human intervention by using a Batch Active Learning based annotation tool. We benchmark different image classification models on the dataset obtained. We have introduced a new Dataset of the images of corn seed with a total of 26,802 images.

paper and code

Deep Web Crawling using Reinforcement Learning (Independent Research Project)

Solving the problem of deep web crawling (which links to follow) using RL to choose next url for crawling and text/image/urls extraction.

New

Build a novel scrapy-based crawler model which uses Reinforcement Learning (A2C, ϵ-greedy) methods to learn links to follow. Start a crawl where reward is given by a classifier that returns a score with 'extracted text' and the 'wiki-page text' match score calculated using cosine similarity after tokenizing.

Corn Seeds Images Dataset

To attract the focus of the vision and machine learning on the application and automation of agriculture, we have introduced a new image dataset.

New Dataset

Build a corn seeds dataset for classification and detection. We use the top and bottom images of seeds as individual images, which can get two independent predictions, increasing the accuracy. To solve the problem of dataset imbalance, we trained a Cond. GAN (BigGAN) to generated fake corn seed images and added them to the dataset. We trained an Active Learning model (BatchBALD) for automatic labelling images and labelled 9K new captured images.
